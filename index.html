 SparkGS

The Gosu JVM Web Microframework

Quick Start

1. Install Gosu

2. Save this code to spark.gsp


    classpath "org.gosu-lang.gosu:spark-gosu:0.10"
    extends sparkgosu.SparkFile

    get('/', \-> "Hello World")

3. Fire it up: $ gosu spark.gsp

4. Check it out at http://localhost:4567/
